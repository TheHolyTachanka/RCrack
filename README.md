## Build From Source

### Prerequisites

* rust

### Building

Clone the repository and use cargo to generate a release build.
```sh
# git clone https://github.com/TheHolyTachanka/RCrack
# cd RCrack/
# cargo build --release
# mv target/release/rcrack
```

## Usage
```sh
# rcrack -d <WORDLIST-FILE-PATH> -h <HASH-TO-CRACK>
```
